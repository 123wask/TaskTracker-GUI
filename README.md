# 📌 TaskTracker

**Консольное и графическое приложение для управления задачами с диагностикой и отчётами**

## 📖 О проекте

TaskTracker – это учебный проект, разработанный в рамках курса «Разработка приложений на C#». Он представляет собой полноценный менеджер задач с **двумя интерфейсами**:

- **Консольный** – для быстрого управления через команды (от 0 до 29).
- **Графический (Windows Forms)** – для удобной повседневной работы с таблицами, вкладками и кнопками.

Приложение хранит все данные локально в формате JSON, не требует интернета и установки (portable). Включает уникальные функции: диагностику системы и сбор отчёта поддержки.

## 💻 Как запустить

### Через Visual Studio
1. Открыть решение `TaskTracker.sln`.
2. Выбрать стартовый проект:
   - `TaskTracker.Gui` – для графического интерфейса.
   - `TaskTracker.App` – для консольного интерфейса.
3. Нажать **F5**.

### Готовая сборка (publish)
- В репозиторий лежит исполняемый файл `TaskTracker.Gui.exe`.
- Просто запустите его – никаких дополнительных установок не требуется.
