Отчёт ЛР23 — расширенный поиск/фильтр

Тема

Реализация расширенного поиска и фильтрации задач

Цель

Научиться реализовывать фильтрацию по нескольким условиям и сохранять настройки пользователя

Что сделано

- Добавлен метод SearchAdvanced(text, status)
- Реализован поиск по Title и Description
- Добавлена фильтрация по статусу
- Добавлены команды:
  - 19) Расширенный фильтр
  - 20) Повтор последнего фильтра
- Последний фильтр сохраняется в config.json

Как проверить

1. Добавить несколько задач
2. Выполнить команду 19 и задать фильтр
3. Убедиться, что выводятся только подходящие задачи
4. Перезапустить программу
5. Выполнить команду 20 и проверить повтор фильтра

Вывод

Реализована расширенная фильтрация с сохранением параметров, что улучшает удобство работы пользователя
