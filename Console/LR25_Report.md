Отчёт ЛР25 — архив задач

Тема

Реализация архива задач

Цель

Научиться скрывать задачи без удаления

Что сделано

- Добавлено поле IsArchived
- Активный список исключает архивные задачи
- Реализованы методы Archive, Unarchive, GetArchive
- Добавлены команды:
  - 24) Архивировать
  - 25) Показать архив
  - 26) Вернуть из архива

Как проверить

1. Добавить задачи
2. Архивировать одну
3. Убедиться, что она исчезла из списка
4. Открыть архив — задача там
5. Вернуть из архива
6. Проверить, что она снова в списке

Вывод

Реализован архив, позволяющий временно скрывать задачи без удаления
