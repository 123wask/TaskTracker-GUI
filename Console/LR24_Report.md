Отчёт ЛР24 — корзина (soft delete)

Тема

Реализация мягкого удаления задач

Цель

Научиться реализовывать soft delete и восстановление данных

Что сделано

- Добавлено поле IsDeleted
- Реализовано удаление в корзину
- Основной список показывает только активные задачи
- Добавлены команды:
  - 21) Показать корзину
  - 22) Восстановить
  - 23) Очистить корзину

Как проверить

1. Добавить задачи
2. Удалить одну
3. Убедиться, что она исчезла из списка
4. Открыть корзину — задача там
5. Восстановить задачу
6. Проверить, что она вернулась

Вывод

Реализована корзина с возможностью восстановления, что предотвращает потерю данных
